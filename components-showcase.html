🗄️ Supabase Database Integration

This library includes built-in Supabase integration for vehicle listings management

// Example: Fetch vehicle listings from Supabase import { VehicleService } from '@smg-automotive/components/lib/supabase'; // Get all live listings const listings = await VehicleService.getAllLive(); // Get specific vehicle by ID const vehicle = await VehicleService.getById('12634083'); // Get vehicles by seller const sellerVehicles = await VehicleService.getBySeller('dealer-123');

📋 Available Database Tables:

📊 listings-db-24jul
Main vehicle listings table
🖼️ images
Vehicle images storage
👥 sellers
Dealer information

🚀 Quick Setup:

  1. Create a .env file in your project root
  2. Add your Supabase credentials:
    NEXT_PUBLIC_SUPABASE_URL=your-project-url NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
  3. Run the connection test: node test-supabase-connection.js
  4. Start building with <VehicleListingsGrid />
